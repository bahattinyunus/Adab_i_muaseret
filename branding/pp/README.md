## Profil Fotoğrafları (PP) — Bahattin Yunus Çetin

Dosyalar 1080x1080 SVG formatındadır (Instagram dairesel kırpma yapar). Güvenli alan merkeze hizalıdır.

- pp_alt1_monogram_serif.svg: Koyu lacivert zemin, altın halka, serif monogram
- pp_alt2_terminal_dark.svg: Terminal teması, <BYÇ> monospaced
- pp_alt3_brackets_engineer.svg: Beyaz minimal, altın parantez yayları
- pp_alt4_hexagon_grid.svg: Hexagon grid + degrade, serif monogram
- pp_alt5_circuit_minimal.svg: Minimal devre çizgileri, iç daire + monogram

Renkler `branding/palette.json` ile uyumludur.

İpucu: SVG'yi Figma/Illustrator/Photopea ile açıp ad/baş harf boyutunu ve harf aralığını (letter-spacing) zevkine göre ayarlayabilirsin. PNG dışa aktarımda 1080x1080 kullan.
